# Eroge Timer v1.0.0

ゲーム画面の手前に時計を表示する、Windows 10/11向けの時計オーバーレイです。

## 主な機能

- ゲームの操作を妨げないクリック透過表示
- 時計と通知領域から開ける右クリックメニュー
- サイズ、位置、フォント、表示形式の変更
- 日付、曜日、秒の表示
- 輪郭線と影
- 代表的なゲーム解像度に合わせた描画
- 設定の自動保存

## 導入方法

1. `eroge_timer-v1.0.0-windows-x64.zip`を展開します。
2. `eroge_timer.exe`を起動します。
3. 時計または通知領域のアイコンを右クリックして設定します。

## 注意事項

- ボーダーレス全画面での使用を推奨します。
- 排他的フルスクリーンでは表示できない場合があります。
- コード署名を行っていないため、Windowsから警告が表示される場合があります。
