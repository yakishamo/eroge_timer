# Changelog

## 1.1.0 - 2026-07-15

### Added

- 日付形式と時刻形式の個別設定、日付と時刻の複数行表示
- 日付または時刻の個別非表示と、不正な両方非表示の防止
- 複数行表示の左揃え、中央揃え、右揃え
- 登録したゲームがアクティブかつ操作中の時間を自動計測
- 無操作、非アクティブ、ゲーム終了時の計測自動停止
- ゲーム登録ごとの測定履歴と過去結果の一覧表示
- 測定履歴のUTF-8 CSV書き出し
- 登録ゲーム内の右クリックによる時計表示切り替え

### Changed

- CSV書き出しをバックグラウンド処理し、ファイル書き込みを効率化
- 保存ダイアログ表示中の低レベルマウスフックを一時停止
- 表示形式の追加に合わせて設定画面を整理

## 1.0.0 - 2026-07-15

初回正式リリース。

### Added

- ゲームより前面に表示するクリック透過時計
- 通知領域と時計上の右クリックメニュー
- 時計のサイズ、位置、フォント、表示形式の設定
- 日付、曜日、秒を含む表示プリセット
- 輪郭線とドロップシャドウ
- ゲーム解像度に合わせた低解像度描画
- `%APPDATA%`への設定保存
- Windowsアプリ固有のアイコン
