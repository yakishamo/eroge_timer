# Eroge Timer v1.1.0

ゲーム画面の手前に時計を表示するWindows 10/11向けオーバーレイです。

## 主な変更

- 日付と時刻の表示形式を個別に設定可能
- 日付と時刻を同時に表示する場合は2行で表示
- 日付または時刻の非表示、文字の左・中央・右揃えに対応
- 登録したゲームのプレイ時間を自動計測
- ゲーム終了、非アクティブ、一定時間の無操作で計測を自動停止
- ゲーム登録ごとに測定履歴を保存し、設定画面で一覧表示
- 測定履歴をUTF-8 CSVとして書き出し
- 登録ゲーム内の右クリックで時計の表示／非表示を切り替え

## 導入方法

1. `eroge_timer-v1.1.0-windows-x64.zip`を展開します。
2. `eroge_timer.exe`を起動します。
3. 時計または通知領域のアイコンを右クリックして設定します。
4. プレイ時間を測定する場合は「プレイ時間計測...」からゲームを登録します。

## v1.0.0からの更新

旧バージョンを終了してから、v1.1.0のファイルへ置き換えてください。既存の設定は引き続き使用でき、従来のプレイ時間データがある場合は測定履歴へ移行されます。

## 注意事項

- ボーダーレス全画面での使用を推奨します。
- 排他的フルスクリーンでは時計を表示できない場合があります。
- アンチチートを使用するゲームでの動作は保証されません。
- プレイ時間は、登録ゲームが前面にあり、無操作時間の上限を超えていない場合に加算されます。
- コード署名を行っていないため、Windowsから警告が表示される場合があります。
